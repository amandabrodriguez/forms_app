# Dev
1.Copiar el .env.template y renombrarlo a .env
2.Cambiar las variables de entorno (The MovieDB(https://www.themoviedb.org/settings/api))
3.Correr el comando:
'''
dart run build_runner build
'''
para generar los archivos necesarios.
